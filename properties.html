CSS Properties Reference

A curated list of commonly used CSS properties with typical values and a short description. Use the search box to filter properties.

display

Values: block | inline | inline-block | flex | grid | none | inline-flex | inline-grid
Sets the element's display type, controlling layout behavior.

position

Values: static | relative | absolute | fixed | sticky
Specifies the positioning method used for an element.

top / right / bottom / left

Values: auto | |
Offsets used with positioned elements to move them from their normal position.

float

Values: left | right | none
Floats an element to the left or right, allowing text to wrap around it (legacy layout).

clear

Values: none | left | right | both | inline-start | inline-end
Specifies which sides of an element where floating elements are not allowed to float.

width / height

Values: auto | | | max-content | min-content
Sets the content width/height of an element.

max-width / max-height

Values: none | |
Limits the maximum size of an element.

min-width / min-height

Values: auto | |
Sets the minimum size an element can shrink to.

box-sizing

Values: content-box | border-box
Controls how width and height are calculated (include border/padding or not).

margin

Values: | | auto
Sets outer spacing around an element (shorthand for margin-top/right/bottom/left).

padding

Values: |
Sets inner spacing between content and border (shorthand for padding-top/right/bottom/left).

border

Values: | none
Shorthand to set border-width, style, and color for all four sides.

border-radius

Values: |
Rounds the corners of an element's outer border edge.

box-shadow

Values: offset-x offset-y blur spread color inset?
Adds shadow effects around an element's frame; can be inset or outset.

color

Values: color keywords | hex | rgb() | rgba() | hsl() | currentColor
Sets the foreground (text) color of an element.

font-family

Values: family-name | generic-family
Specifies prioritized list of font family names and/or generic family names.

font-size

Values: | | |
Specifies the size of the font.

font-weight

Values: normal | bold | bolder | lighter | 100..900
Sets the weight (boldness) of the font.

line-height

Values: normal | | |
Sets the height of a line box; controls spacing between lines of text.

text-align

Values: left | right | center | justify | start | end
Specifies horizontal alignment of inline content inside a block.

text-decoration

Values: none | underline | overline | line-through | blink | color | style
Shorthand for setting decorations like underline and color/ style.

background

Values: color | image | position | size | repeat | attachment | clip | origin
Shorthand for background-color, image, position, size, repeat, and attachment.

background-image

Values: url() | linear-gradient() | radial-gradient() | none
Sets one or more background images for an element.

background-size

Values: auto | cover | contain | |
Specifies the size of the background images.

opacity

Values: 0.0 - 1.0
Sets the transparency level of an element (and its children).

filter

Values: none | blur() | brightness() | contrast() | drop-shadow() | hue-rotate() | saturate()
Applies graphical effects like blur and color shifts to an element.

flex

Values: none | auto | |
Shorthand for flex-grow, flex-shrink and flex-basis on flex items.

justify-content

Values: flex-start | flex-end | center | space-between | space-around | space-evenly
Defines alignment along the main axis in flex or grid containers.

align-items

Values: stretch | flex-start | flex-end | center | baseline
Aligns flex items along the cross axis (vertical by default).

grid-template-columns / rows

Values: | repeat() | auto | auto-fit | auto-fill
Defines the columns (or rows) of the grid container.

transition

Values:
Shorthand to animate changes to CSS properties smoothly.

animation

Values: name duration timing-function delay iteration-count direction fill-mode
Shorthand for CSS animations using @keyframes.

cursor

Values: auto | default | pointer | move | text | wait | not-allowed | url()
Specifies the mouse cursor displayed when pointing over an element.

overflow

Values: visible | hidden | scroll | auto
Controls what happens to content that overflows an element's box.

visibility

Values: visible | hidden | collapse
Controls whether an element is visible, but still takes up space when hidden.

z-index

Values: auto |
Controls the stacking order of positioned elements.
