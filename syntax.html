CSS Syntax Reference

This page shows the actual CSS syntax for the concepts demonstrated on the home page. Each section contains a short explanation, a code example, and a live preview so you can see the styles in action.

Custom Properties (Variables)

Define variables inside :root and reuse them throughout your stylesheet for consistency.

:root {
  --accent: #ffd54f;
  --muted: #9aa4b2;
}

.button {
  background: var(--accent);
  color: #000;
  padding: .6rem 1rem;
}
Accent background using var(--accent)
This box uses var(--muted) for muted text color

Selectors

Common selectors: element, class, id, attribute, pseudo-classes and pseudo-elements.

/* element */
button { padding:.6rem }

/* class */
.btn { background:#ffd54f }

/* id */
#main { max-width:1200px }

/* attribute */
a[target] { text-decoration:underline }

/* pseudo */
a:hover { opacity:.9 }
a:hover (hover me)

Box Model

Content, padding, border and margin form the box model. Use DevTools to inspect spacing.

.card {
  padding: 1rem; /* space inside */
  border: 1px solid #222; /* border */
  margin: 1rem; /* outer spacing */
}
Content area (padding + border)

Flexbox

Flexbox lets you align items along a single axis easily.

.row {
  display: flex;
  gap: 1rem;
  align-items: center;
}

.item { flex: 1 }

CSS Grid

Grid is powerful for two-dimensional layouts.

.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
  gap: 1rem;

Transforms

Use transform to rotate, scale, skew and translate elements.

.spin { transform: rotate(45deg); }
.scale { transform: scale(1.15); }
.translate { transform: translateX(10px); }

Gradients

Linear, radial and conic gradients create smooth color transitions.

background: linear-gradient(135deg,#ffd54f,#ffb300);
background: radial-gradient(circle,#1abc9c,#16a085);
background: conic-gradient(from 45deg,#9b59b6,#e74c3c,#f39c12);

Buttons

Button patterns: primary, secondary, and icon buttons.

.btn-primary{ background:var(--accent); color:#000 }
.btn-secondary{ border:1px solid var(--accent); background:transparent }
